# Sentinel - Multilingual Hate Speech Detection Platform

Sentinel is a premium, high-performance web interface designed for content moderation. 
It supports real-time multilingual hate speech classification in English and Hindi, powered by a fine-tuned Transformer model.

## Features
- **Modern AMOLED Black Design**: High-contrast, McLaren-inspired aesthetics with glassmorphic cards and dynamic orange glows.
- **Multilingual Analysis**: Real-time hate speech detection with detailed class probabilities for English and Hindi text.
- **Interactive UI**: Fluid micro-animations, customizable dark elements, and performance metrics.
